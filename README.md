# 🎴 ANIMEX - Odisea Moderna del Anime

**ANIMEX** es una plataforma minimalista y para descubrir anime. Utiliza datos en tiempo real desde la API de Jikan (MyAnimeList) para ofrecer una experiencia rápida, clara y centrada en el contenido.

---

La aplicación presenta una paleta estrictamente minimalista en escala de grises (negro, blanco y grises), enfocándose en:

- Tipografía de alto contraste  
- Diseños editoriales limpios  
- Protagonismo total del arte del anime  


📱 **Navegación Responsive**  
Barra de navegación flotante optimizada para móviles y un header limpio para una navegación fluida.

---

## 🛠️ Stack

### Frontend

- React 19 
- TypeScript   
- Tailwind CSS   
- React Router DOM 
- Lucide React   


## 🛠️ Instalación y Configuración

1. Clonar el repositorio:

```bash
git clone <tu-repo>
```

2. Instalar dependencias:

```bash
npm install
```

3. Ejecutar el servidor de desarrollo:

```bash
npm run dev
```

4. Build de producción:

```bash
npm run build
```

---

## 📜 Scripts

- `npm run dev` → Inicia el servidor de desarrollo (Vite)  
- `npm run build` → Genera el build de producción  
- `npm run lint` → Ejecuta el chequeo de TypeScript  
- `npm run preview` → Previsualiza el build de producción
